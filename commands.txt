 I1 : invert matrix 1
 I2 : invert matrix 2
 L  : list current cell parameters, matrices, and vectors
 M1 : read matrix 1
 M2 : read matrix 2
 MA : apply matrix 1 to current cell parameters
 MI : interchange matrix 1 and matrix 2
 MM : matrix 1 * matrix 2
 MR : reset matrix 1 to 1
 MV : matrix * vector
 V  : read vector
 VM : vector * matrix
 VS : vector * vector (scalar)
 VV : vector * vector (vector)

 
Appendix 2:  commands for the RPN calculator
 .  : enter to register 1 (x), scroll enabled 
 .. : shift stack, keep register 1 (register 4 is lost) 
 .1 : enter to reg. 1  (x) (override) 
 .2 : enter to reg. 2  (y) (override) 
 .3 : enter to reg. 3 (override) 
 .4 : enter to reg. 4 (override) 
 .M : enter to memory reg. (override)
 <  : roll stack left (down)
 >  : roll stack right (up)
 <> : interchange register 1 and 2 (x and y)
 +  : +    (y + x)   (reg.2 + reg.1) 
 -  : -    (y ľ x)    (reg.2 ľ reg.1) 
 *  : *   (y Ľ x)      (reg.2 Ľ reg.1) 
 / (also : ) :  /   (y / x)      (reg.2 / reg.1)
 ** : **    y**x  (reg.2 ** reg.1)
 !  : use content of RPN-register 1 as input value 
 !1 : use content of RPN-register 1 as input value (same as !  )
 !2 : use content of RPN-register 2 as input value
 !3 : use content of RPN-register 3 as input value
 !4 : use content of RPN-register 4 as input value
 !M : use content of RPN-memory as input value
 AC : acos
 AS : asin 
 AT : atan
 CO : cos 
 CS : change sign 
 E* : exp   e**x   (ex )
 LO : ln, (logarithm, base e)
 LX : last x (RPN)
 MP : put content of reg.1 to memory (RPN)
 MG : get content of memory (put to reg.1) (RPN)
 RE : 1/x  (reciprocal) (RPN)
 RT : square root (RPN)
 SI : sin (RPN)
 SQ : x2  (square)  (RPN)
 TG : tangens (RPN)
